6.3  LP Formulation (Develop in Colab, Run Locally)

Objective: minimize sum(tariff[t] × load[t]) for t in range(48)
Constraint 1 (equality): sum(load[t]) == total_demand — demand must be met
Constraint 2 (bounds): each load[t] must be within [max(0, forecast[t] - flex_limit), min(capacity, forecast[t] + flex_limit)]
Where flex_limit = forecast[t] × flexibility_fraction (start with 0.20 = 20%)
Use scipy.optimize.linprog with method='highs'
Check result.status == 0 before using the result. Status 2 = infeasible. Handle it gracefully.
Compute: original_cost = sum(tariff × forecast), optimized_cost = sum(tariff × result.x), savings = original - optimized
